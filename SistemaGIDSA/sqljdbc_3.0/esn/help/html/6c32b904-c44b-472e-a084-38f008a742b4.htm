Método setBinaryStream (int, java.io.InputStream)

Establece el parámetro designado en el flujo de entrada especificado.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public final void setAsciiStream(int parameterIndex,
                                 java.io.InputStream x)

Parámetros

parameterIndex

Un valor int que indica el número del parámetro.

x

Un objeto java.io.InputStream.

Excepciones

SQLServerException

Notas

El método setBinaryStream especifica este método setBinaryStream en la interfaz java.sql.PreparedStatement.
