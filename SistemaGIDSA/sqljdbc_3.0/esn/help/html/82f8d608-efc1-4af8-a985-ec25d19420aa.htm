Método registerOutParameter (SQLServerCallableStatement)

Registra el parámetro OUT.

Lista de sobrecargas

Nombre Descripción

registerOutParameter (int, int)

Registra el parámetro OUT en la posición ordinal especificada para el tipo JDBC determinado.

registerOutParameter (int, int, int)

Registra el parámetro OUT en la posición ordinal especificada para el tipo JDBC y escala determinados.

registerOutParameter (int, int, java.lang.String)

Registra el parámetro OUT en la posición ordinal especificada para el tipo JDBC y el nombre del tipo especificados.

registerOutParameter (java.lang.String, int)

Registra el parámetro OUT con el nombre especificado para el tipo JDBC especificado.

registerOutParameter (java.lang.String, int, int)

Registra el parámetro OUT con el nombre especificado para el tipo JDBC y la escala determinados.

registerOutParameter (java.lang.String, int, java.lang.String)

Registra el parámetro OUT con el nombre especificado para el nombre de tipo y el tipo JDBC dado.
