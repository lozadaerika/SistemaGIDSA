Método absolute (ISQLServerResultSet)

Mueve el cursor a la fila especificada en este objeto SQLServerResultSet.

Sintaxis

public boolean absolute(int row)

Parámetros

row

Un valor int que indica el número de fila al que se va a desplazar. Puede ser positivo, negativo o 0.

Valor devuelto

Es true si el cursor se mueve a la posición determinada. Es false si el cursor está antes de la primera fila o después de la última fila.

Excepciones

SQLServerException

Notas

El método absolute especifica este método absolute en la interfaz java.sql.ResultSet.
