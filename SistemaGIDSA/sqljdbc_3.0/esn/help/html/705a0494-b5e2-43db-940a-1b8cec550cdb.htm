Método setSendTimeAsDatetime (SQLServerDataSource)

Este método se agregó al controlador JDBC 3.0 de SQL Server.

Modifica el valor de la propiedad de conexión sendTimeAsDatetime.

Sintaxis

public void setSendTimeAsDatetime(boolean sendTimeAsDateTime)

Parámetros

sendTimeAsDateTime

Un valor booleano. Cuando sea TRUE, produce valores java.sql.Time que se envían al servidor como tipos datetime de SQL Server. Cuando sea FALSE, produce valores java.sql.Time que se envían al servidor como tipos time de SQL Server.

Notas

SQLServerDataSource.getSendTimeAsDatetime devuelve el valor de configuración de la propiedad de conexión sendTimeAsDatetime.

Para obtener más información acerca de la propiedad de conexión sendTimeAsDatetime, vea Establecer las propiedades de conexión.

Para obtener más información, vea Configurar el modo en que los valores java.sql.Time se envían al servidor.
