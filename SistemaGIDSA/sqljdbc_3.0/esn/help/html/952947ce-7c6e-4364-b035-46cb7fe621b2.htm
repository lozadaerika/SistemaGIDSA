updateDateTimeOffset(string, microsoft.sql.DateTimeOffset) (ISQLServerResultSet)

Este método se agregó en el controlador JDBC 3.0 de Microsoft SQL Server.

Actualiza el valor de la columna especificada en el valor Clase DateTimeOffset, según un nombre de columna.

Sintaxis

public void updateDateTimeOffset(String columnName, microsoft.sql.DateTimeOffset x)

Parámetros

columnName

El nombre de una columna.

x

Un objeto Clase DateTimeOffset.

Excepciones

SQLServerException

Notas

Puede recuperar un valor Clase DateTimeOffset con SQLServerResultSet.getDateTimeOffset.
