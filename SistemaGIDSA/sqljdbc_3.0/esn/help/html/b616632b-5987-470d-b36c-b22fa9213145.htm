Clase SQLServerDataSourceObjectFactory

Representa un servicio de generación de objetos para materializar los orígenes de datos de la interfaz Java Naming and Directory Interface (JNDI).

Paquete: com.microsoft.sqlserver.jdbc

Extiende: java.lang.Object

Implementa: javax.naming.spi.ObjectFactory

Sintaxis

public class SQLServerDataSourceObjectFactory

Notas

Este método lo heredan todas las clases de orígenes de datos. El controlador JDBC de Microsoft SQL Server expone esta clase que implementa un objeto ObjectFactory, como parte de la compatibilidad que ofrece a la interfaz Referenceable. Cuando los servidores de la aplicación Java llamen a getReference en una clase de origen de datos, esto creará un objeto Reference que utilizará internamente el nombre de la case como su servicio de generación de clases.

Cuando el servidor de la aplicación Java tenga que eliminar referencias del objeto Reference, crea una instancia del objeto SQLServerDataSourceObjectFactory y llama al método getObjectInstance para pasar el objeto Reference y recuperar la instancia del origen de datos.
