Método setObject (java.lang.String, java.lang.Object, int)

Establece el valor del parámetro designado mediante el uso del objeto y tipo de destino determinados.

Sintaxis

public void setObject(java.lang.String sCol,
                      java.lang.Object o,
                      int n)

Parámetros

sCol

Un separador String que contiene el nombre del parámetro.

o

Un valor Object .

n

Un valor int que indica el tipo de destino tal como queda definido en java.sql.Types.

Excepciones

SQLServerException

Notas

El método setObject especifica este método setObject en la interfaz java.sql.CallableStatement.

Desde el controlador JDBC 3.0 de SQL Server, el comportamiento del método queda modificado por la propiedad de conexión sendTimeAsDatetime (Establecer las propiedades de conexión) y SQLServerDataSource.setSendTimeAsDatetime.

Para obtener más información, vea Configurar el modo en que los valores java.sql.Time se envían al servidor.
