Método getResponseBuffering (SQLServerDataSource)

Devuelve el método de almacenamiento en búfer de respuesta para este objeto SQLServerDataSource.

Sintaxis

public java.lang.String getResponseBuffering()

Valor devuelto

Un String que contiene un full o adaptive en minúsculas.

Notas

El valor full se especifica mediante la lectura del resultado completo desde el servidor en el tiempo de ejecución.

El valor adaptive especifica que se almacenen en búfer los menos datos posibles cuando sea necesario. El valor adaptive es el modo de almacenamiento en búfer predeterminado.

Para obtener más información sobre cómo utilizar el modo de almacenamiento en búfer de respuesta, vea Usar el almacenamiento en búfer adaptable.
