Método removeConnectionEventListener (SQLServerPooledConnection)

Quita la escucha del evento determinado.

Sintaxis

public void removeConnectionEventListener(javax.sql.ConnectionEventListener listener)

Parámetros

listener

Un objeto ConnectionEventListener.

Notas

El método removeConnectionEventListener es especificado por el método removeConnectionEventListener en la interfaz javax.sql.PooledConnection.
