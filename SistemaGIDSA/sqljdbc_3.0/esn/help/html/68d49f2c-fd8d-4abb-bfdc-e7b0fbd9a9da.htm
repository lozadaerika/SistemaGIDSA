Método setClob (int, java.sql.Clob)

Establece el parámetro designado para el objeto Clob determinado.

Sintaxis

public final void setClob(int parameterIndex,
                          java.sql.Clob clobValue)

Parámetros

parameterIndex

Un valor int que indica el número del parámetro.

clobValue

Un objeto Clob.

Excepciones

SQLServerException

Notas

El método setClob especifica este método setClob en la interfaz java.sql.PreparedStatement.
