Mejorar el rendimiento y la confiabilidad con el controlador JDBC

Un aspecto del desarrollo de aplicaciones común a todas las aplicaciones es la necesidad constante de mejorar el rendimiento y la confiabilidad. Hay una serie de técnicas para hacer esto con el controlador JDBC de Microsoft SQL Server.

En los temas de esta sección se describen las distintas técnicas para aumentar el rendimiento y la confiabilidad de la aplicación al usar el controlador JDBC.
