Método getSQLXML (int)

Recupera el valor del parámetro designado como un objeto SQLXML según el índice del parámetro.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public final java.sql.SQLXML getSQLXML(int parameterIndex)

Parámetros

parameterIndex

Un valor int que indica el índice del parámetro.

Valor devuelto

Un objeto SQLXML .

Excepciones

SQLServerException

Notas

El método getSQLXML especifica este método getSQLXML en la interfaz java.sql.CallableStatement.
