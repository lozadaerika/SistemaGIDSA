Método setServerName (SQLServerDataSource)

Establece el nombre del equipo que ejecuta SQL Server.

Sintaxis

public void setServerName(java.lang.String serverName)

Parámetros

serverName

Un String que contiene el nombre de servidor.

Notas

El nombre del servidor es el nombre de host del equipo de destino que ejecuta SQL Server. Si no se establece la propiedad serverName, getServerName, devuelve el valor predeterminado que es NULL.
