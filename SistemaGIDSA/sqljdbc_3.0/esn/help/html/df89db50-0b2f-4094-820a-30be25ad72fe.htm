Método getMoreResults ()

Se desplaza al próximo resultado de este objeto SQLServerStatement.

Sintaxis

public final boolean getMoreResults()

Valor devuelto

Es true si el resultado devuelto es un conjunto de resultados. En caso contrario, es false.

Excepciones

SQLServerException

Notas

El método getMoreResults especifica este método getMoreResults en la interfaz java.sql.Statement.

Al llamar implícitamente al método getMoreResults, se cierra los objetos de conjunto de resultados que estén abiertos en esos momentos y que hubieran obtenido con el método getResultSet.
