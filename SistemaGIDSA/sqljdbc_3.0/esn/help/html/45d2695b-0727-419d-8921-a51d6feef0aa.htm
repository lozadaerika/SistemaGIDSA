Método getNCharacterStream (java.lang.String)

Recupera el valor del parámetro designado como un objeto Reader según el nombre de parámetro.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public final java.io.Reader getNCharacterStream(java.lang.String columnLabel)

Parámetros

columnLabel

Un String que contiene la etiqueta de columna.

Valor devuelto

Un objeto Reader .

Excepciones

SQLServerException

Notas

Este método se debe utilizar cuando se tenga acceso a los parámetros NCHAR, LONGNVARCHAR y NVARCHAR.

El método getNCharacterStream especifica este método getNCharacterStream en la interfaz java.sql.CallableStatement.
