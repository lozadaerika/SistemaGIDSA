Método getFloat (java.lang.String)

Recupera el valor del parámetro designado como un valor float en el lenguaje de programación Java según el nombre del parámetro.

Sintaxis

public float getFloat(java.lang.String sCol)

Parámetros

sCol

Un String que contiene el nombre del parámetro.

Valor devuelto

Un valor float .

Excepciones

SQLServerException

Notas

El método getFloat es especificado por el método getFloat en la interfaz java.sql.CallableStatement.

Este método devuelve todos los tipos de datos basados en número con fidelidad float de Java.
