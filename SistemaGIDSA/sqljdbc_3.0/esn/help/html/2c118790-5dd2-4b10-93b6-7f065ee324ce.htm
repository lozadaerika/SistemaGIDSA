Método getColumnClassName (SQLServerResultSetMetaData)

Devuelve el nombre completo de la clase Java cuyas instancias se generan si se llama al método getObject de la clase SQLServerResultSet para que recupere un valor en la columna.

Sintaxis

public java.lang.String getColumnClassName(int column)

Parámetros

column

Un valor int que indica el índice de la columna.

Valor devuelto

Un String que contiene el nombre completo de la clase.

Excepciones

SQLServerException

Notas

El método getColumnClassName especifica este método getColumnClassName en la interfaz java.sql.ResultSetMetaData.
