Método getNCharacterStream (int) (ISQLServerResultSet)

Recupera el valor de una columna designada en la fila actual del objeto SQLServerResultSet como un objeto Reader.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public java.io.Reader getNCharacterStream(int columnIndex)

Parámetros

columnIndex

Un valor int que indica el índice de la columna.

Valor devuelto

Un objeto Reader.

Excepciones

SQLServerException

Notas

El método getNCharacterStream especifica este método getNCharacterStream en la interfaz java.sql.ResultSet.

Este método se puede utilizar para recuperar el valor de una columna nvarchar, nchar, nvarchar(max), ntext o xml en la fila actual de este objeto SQLServerResultSet. Si intenta utilizar este método para recuperar valores de otros tipos de datos, se producirá una excepción.
