Método setResponseBuffering (SQLServerDataSource)

Establece el modo de almacenamiento en búfer de respuesta para las conexiones creadas mediante este objeto SQLServerDataSource.

Sintaxis

public void setResponseBuffering(java.lang.String value)

Parámetros

value

Un String que contiene el modo de almacenamiento en búfer y de transmisión de datos. El modo válido puede ser uno de los siguientes valores String que no distinguen mayúsculas de minúsculas: full o adaptive.

Notas

El valor full se especifica mediante la lectura del resultado completo desde el servidor en el tiempo de ejecución.

El valor adaptive especifica que se almacenen en búfer los menos datos posibles cuando sea necesario. El valor adaptive es el modo de almacenamiento en búfer para la versión 2.0 del controlador JDBC de Microsoft SQL Server o posterior.

Para obtener más información sobre cómo utilizar el modo de almacenamiento en búfer de respuesta, vea Usar el almacenamiento en búfer adaptable.
