Método getCatalog (ISQLServerConnection)

Recupera el nombre del catálogo actual de este objeto SQLServerConnection.

Sintaxis

public java.lang.String getCatalog()

Valor devuelto

Un separador String que contiene el nombre del catálogo.

Excepciones

SQLServerException

Notas

El método getCatalog especifica este método getCatalog en la interfaz java.sql.Connection.

Devuelve la propiedad de catálogo actual del objeto SQLServerConnection, o bien NULL se no se ha establecido. La propiedad de catálogo se establece explícitamente con el método setCatalog o se actualiza implícitamente mediante la lectura del cambio del entorno en el TDS correspondiente al catálogo actual.
