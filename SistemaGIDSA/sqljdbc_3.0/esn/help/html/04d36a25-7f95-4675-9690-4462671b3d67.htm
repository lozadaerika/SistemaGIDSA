Método prepareCall (java.lang.String, int, int)

Crea un objeto SQLServerCallableStatement que genera objetos SQLServerResultSet con el tipo y simultaneidad especificados.

Sintaxis

public java.sql.CallableStatement prepareCall(java.lang.String sql,
                                              int resultSetType,
                                              int resultSetConcurrency)

Parámetros

sql

Un String que contiene una instrucción SQL.

resultSetType

Un valor int que indica el tipo de conjunto de resultados.

resultSetConcurrency

Un valor int que indica el tipo de simultaneidad del conjunto de resultados.

Valor devuelto

Un objeto CallableStatement.

Excepciones

SQLServerException

Notas

El método prepareCall especifica este método prepareCall en la interfaz java.sql.Connection.
