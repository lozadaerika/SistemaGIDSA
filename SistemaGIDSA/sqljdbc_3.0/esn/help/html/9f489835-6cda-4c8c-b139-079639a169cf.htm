Método setResponseBuffering (ISQLServerStatement)

Establece el modo de almacenamiento en búfer de respuesta para este objeto SQLServerStatement en String full con distinción entre mayúsculas y minúsculas o en adaptive.

Sintaxis

public final void setResponseBuffering(java.lang.String value)

Parámetros

value

Un String que contiene el modo de almacenamiento en búfer de respuesta. El modo válido puede ser uno de los siguientes valores String que no distinguen mayúsculas de minúsculas: full o adaptive.

Excepciones

SQLServerException

Notas

El valor adaptive especifica que se almacenen en búfer los menos datos posibles cuando sea necesario.

El valor full se especifica mediante la lectura del resultado completo desde el servidor en el tiempo de ejecución.

El valor adaptative es el valor predeterminado en la versión 2.0 y 3.0 de JDBC. El valor full era el valor predeterminado en la versión 2.0 del controlador JDBC.

El método setResponseBuffering le permite invalidar la propiedad String de la conexión responseBuffering para el objeto SQLServerStatement actual. Para obtener más información sobre cómo utilizar el modo de almacenamiento en búfer de respuesta, vea Usar el almacenamiento en búfer adaptable.

Si la aplicación especifica un valor de parámetro no válido para el método setResponseBuffering, se produce una excepción SQLServerException.
