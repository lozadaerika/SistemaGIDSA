Método getObject (ISQLServerResultSet)

Obtiene el valor de la columna designada en la fila actual de este objeto SQLServerResultSet como un objeto en el lenguaje de programación Java.

Lista de sobrecargas

Nombre Descripción

getObject (int)

Obtiene el valor del índice de la columna designada en la fila actual de este objeto SQLServerResultSet como un objeto en el lenguaje de programación Java.

getObject (int, java.util.Map)

Obtiene el valor del índice de columna designada en la fila actual de este objeto SQLServerResultSet como un objeto en el lenguaje de programación Java, para ello usa el objeto Map determinado.

getObject (java.lang.String)

Obtiene el valor del nombre de la columna designada en la fila actual de este objeto SQLServerResultSet como un objeto en el lenguaje de programación Java.

getObject (java.lang.String, java.util.Map)

Obtiene el valor del nombre de columna designada en la fila actual de este objeto SQLServerResultSet como un objeto en el lenguaje de programación Java, para ello usa el objeto Map determinado.
