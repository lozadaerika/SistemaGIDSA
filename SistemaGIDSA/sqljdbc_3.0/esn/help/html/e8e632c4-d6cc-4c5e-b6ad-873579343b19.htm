Método getClientInfo (java.lang.String)

Recupera el valor de una propiedad de información del cliente especificada.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public java.lang.String getClientInfo (java.lang.String name)

Parámetros

name

Un String que contiene el nombre de la propiedad de información de cliente que se va a recuperar.

Valor devuelto

Un String que contiene el valor de la propiedad de información de cliente.

Excepciones

SQLServerException

Notas

El método getClientInfo especifica este método getClientInfo en la interfaz java.sql.Connection.

El controlador JDBC de Microsoft SQL Server no admite propiedades de información de cliente. Como resultado, este método devuelve null.

De igual forma, las aplicaciones pueden utilizar el método getClientInfoProperties de la clase SQLServerDatabaseMetaData para recuperar una lista de las propiedades de información de cliente que admita el controlador. El método getClientInfoProperties devuelve un conjunto de resultados vacío.
