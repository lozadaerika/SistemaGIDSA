Método setDatabaseName (SQLServerDataSource)

Establece el nombre de la base de datos a la que se va a efectuar la conexión.

Sintaxis

public void setDatabaseName(java.lang.String databaseName)

Parámetros

databaseName

Un String que contiene el nombre de la base de datos.
