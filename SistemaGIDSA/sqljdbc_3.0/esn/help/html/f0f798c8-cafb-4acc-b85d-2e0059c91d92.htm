Método setPoolable (ISQLServerStatement)

Solicita que una instrucción se agrupe o no.

Sintaxis

public void setPoolable(boolean poolable) throws SQLException

Parámetros

poolable

Si es true, solicita que la instrucción se agrupe. Si es false, solicita que la instrucción no se agrupe.

Excepciones

SQLServerException

Notas

El valor especificado en el parámetro poolable es una sugerencia de implementación para el grupo de instrucciones que indica si la aplicación quiere que se agrupe la instrucción. El administrador del grupo de instrucciones decide si va a utilizar la sugerencia.

El valor del grupo de una instrucción se aplica a las memorias caché de instrucciones internas que implementa el controlador y a las memorias caché de instrucciones externas que implementan los servidores y otras aplicaciones.

De forma predeterminada, un objeto SQLServerStatement no se puede agrupar cuando se crea. Los objetos SQLServerCallableStatement y SQLServerPreparedStatement se pueden agrupar cuando se crean.

Se produce una excepciónSQLServerException si se llama a este método en una instrucción cerrada.

isPoolable devuelve un valor que indica si se puede agrupar el objeto.
