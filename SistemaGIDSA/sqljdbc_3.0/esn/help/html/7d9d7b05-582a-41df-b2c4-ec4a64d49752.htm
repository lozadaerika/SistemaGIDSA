Método getDateTimeOffset (SQLServerCallableStatement)

Este método se agregó en el controlador JDBC 3.0 de Microsoft SQL Server.

Recupera el valor del parámetro designado como un objeto Clase DateTimeOffset en el lenguaje de programación Java según el índice del parámetro.

Puede establecer un valor de parámetro Clase DateTimeOffset con SQLServerCallableStatement.setDateTimeOffset.
