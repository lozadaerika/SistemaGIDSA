Método executeUpdate (java.lang.String, java.lang.String[])

Ejecuta la instrucción SQL determinada e indica al controlador JDBC de Microsoft SQL Server que las claves que se han generado automáticamente y están presentes en la matriz determinada deben estar disponibles para su recuperación.

Sintaxis

public final int executeUpdate(java.lang.String sql,
                               java.lang.String[] columnNames)

Parámetros

sql

Un String que contiene una instrucción SQL.

columnNames

Una matriz de String que indica que los nombres de columna de las claves que se generaron automáticamente deben estar disponibles.

Valor devuelto

Un valor int que indica el número de filas afectadas; 0 si se utiliza una instrucción DDL.

Excepciones

SQLServerException

Notas

El método executeUpdate especifica este método executeUpdate en la interfaz java.sql.Statement.
