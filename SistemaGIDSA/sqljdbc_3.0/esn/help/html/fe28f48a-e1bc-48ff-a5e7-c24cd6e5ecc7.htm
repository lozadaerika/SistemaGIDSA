Usar instrucciones con SQL

Si trabaja con los datos de una base de datos de SQL Server con el controlador JDBC de Microsoft SQL Server e instrucciones SQL insertadas, haya varias clases que puede usar. La clase usada depende del tipo de instrucción SQL que desee ejecutar.

Si la instrucción SQL no contiene parámetros IN, use la clase SQLServerStatement, pero si contiene parámetros IN, use la clase SQLServerPreparedStatement.

Nota: Si necesita usar instrucciones SQL que contengan parámetros IN y OUT, tendrá que implementarlas como un procedimiento almacenado y llamarlo con la clase SQLServerCallableStatement. Para obtener más información sobre el uso de procedimientos almacenados, consulte Usar instrucciones con procedimientos almacenados.
