Método getBytes (int) (ISQLServerResultSet)

Recupera el valor del índice de la columna designada en la fila actual de este objeto SQLServerResultSet como una matriz byte en el lenguaje de programación Java.

Sintaxis

public byte[] getBytes(int columnIndex)

Parámetros

columnIndex

Un valor int que indica el índice de la columna.

Valor devuelto

Una matriz de valores byte .

Excepciones

SQLServerException

Notas

El método getBytes especifica este método getBytes en la interfaz java.sql.ResultSet.

Este método admite la recuperación de todas las columnas como una lectura de bytes sin formato desde el servidor. Devuelve una matriz de bytes directamente desde el servidor, en el formato que está almacenado en el servidor.

En el controlador JDBC 2.0 de Microsoft SQL Server, podía utilizar SQLServerResultSet.getBytes para convertir valores entre las matrices de bytes y los tipos de datos date, time, datetime2 o datetimeoffset de SQL Server. Al utilizar este método con esos tipos de datos, en el controlador JDBC 3.0 de SQL Server, se producirá una excepción que indica que no se admite la conversión.
