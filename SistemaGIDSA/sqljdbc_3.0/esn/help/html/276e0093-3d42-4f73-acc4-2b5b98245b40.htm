Método getXAConnection (java.lang.String, java.lang.String)

Intenta establecer una conexión a bases de datos física mediante el nombre de usuario y contraseña especificados.

Sintaxis

public javax.sql.XAConnection getXAConnection(java.lang.String user,
                                              java.lang.String password)

Parámetros

user

Un String que contiene el nombre de usuario.

password

Un String que contiene la contraseña.

Valor devuelto

Un objeto XAConnection.

Excepciones

java.sql.SQLException

Notas

El método getXAConnection especifica este método getXAConnection en la interfaz javax.sql.XADataSource.

Nota: Las implementaciones del grupo de conexiones XA, y no los códigos de aplicación JDBC, llaman normalmente a este método.
