Método getClientInfo (ISQLServerConnection)

Recupera información con respecto a las propiedades de información del cliente que admita el controlador JDBC.

Nombre Descripción

Método getClientInfo ()

Recupera una lista que contiene el nombre y valor vigentes de cada propiedad de información del cliente que admita el controlador JDBC.

Método getClientInfo (java.lang.String)

Recupera el valor de una propiedad de información del cliente especificada.
