Método getColumnTypeName (SQLServerResultSetMetaData)

Recupera el nombre del tipo específico de base de datos de la columna designada.

Sintaxis

public java.lang.String getColumnTypeName(int column)

Parámetros

column

Un valor int que indica el índice de la columna.

Valor devuelto

Un String que contiene el nombre de servidor de la columna.

Excepciones

SQLServerException

Notas

El método getColumnTypeName especifica este método getColumnTypeName en la interfaz java.sql.ResultSetMetaData.

El controlador JDBC 3.0 de Microsoft SQL Server tiene cambios de comportamiento en la columna TYPE_NAME. Vea SQLServerDatabaseMetaData.getColumns para obtener más información.
