Método setApplicationName (SQLServerDataSource)

Establece el nombre de la aplicación.

Sintaxis

public void setApplicationName(java.lang.String applicationName)

Parámetros

applicationName

Un objeto String que contiene el nombre de la aplicación.

Notas

El nombre de la aplicación se usa para identificar la aplicación específica en diversas herramientas de creación de perfiles y registros de SQL Server. Si no se establece el nombre de aplicación, el método getApplicationName devuelve la cadena no localizada "controlador JDBC de Microsoft SQL Server".
