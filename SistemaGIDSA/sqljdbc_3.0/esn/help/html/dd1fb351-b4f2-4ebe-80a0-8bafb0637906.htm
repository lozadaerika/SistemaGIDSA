Método length (SQLServerNClob)

Recupera el número de caracteres del objeto NClob.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public long length()

Valor devuelto

La longitud de NClob en número de caracteres.

Excepciones

SQLServerException

Notas

El método length especifica este método length en la interfaz java.sql.NClob.
