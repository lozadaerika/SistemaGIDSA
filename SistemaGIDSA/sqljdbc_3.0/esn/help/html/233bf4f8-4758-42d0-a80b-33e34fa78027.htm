Método setFetchSize (ISQLServerResultSet)

Ofrece al controlador JDBC una sugerencia sobre el número de filas que se deberían capturar en la base de datos cuando se necesitan más filas para este objeto SQLServerResultSet.

Sintaxis

public void setFetchSize(int rows)

Parámetros

rows

Un valor int que indica el número de filas que se van a capturar.

Excepciones

SQLServerException

Notas

El método setFetchSize especifica este método setFetchSize en la interfaz java.sql.ResultSet.

Si el tamaño de la captura es cero, el controlador JDBC omite el valor y calcula el tamaño de captura apropiado. El objeto SQLServerStatement que creó el conjunto de resultados establece el valor predeterminado. El tamaño de la captura puede cambiar en cualquier momento.

Este método cambia el tamaño de la captura del bloque para los cursores de servidor y surte efecto la próxima vez que el controlador JDBC necesite llamar a sp_cursorfetch. Al establecer el tamaño de la captura en cero, se restaura el tamaño predeterminado de la misma para el tipo de cursor que se esté utilizando en esos momentos.
