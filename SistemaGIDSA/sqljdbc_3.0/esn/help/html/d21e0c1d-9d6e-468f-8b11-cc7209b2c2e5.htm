Método getTime (int, java.util.Calendar) (ISQLServerResultSet)

Recupera el valor del índice de la columna designado en la fila actual de este objeto SQLServerResultSet como un objeto java.sql.Time en el lenguaje de programación Java, para ello, usa el objeto Calendar determinado.

Sintaxis

public java.sql.Time getTime(int columnIndex,
                             java.util.Calendar cal)

Parámetros

columnIndex

Un valor int que indica el índice de la columna.

cal

Un objeto Calendar.

Valor devuelto

Un objeto Time.

Excepciones

SQLServerException

Notas

El método getTime especifica este método getTime en la interfaz java.sql.ResultSet.

Este método devuelve una hora válida que forma parte de un tipo de datos datetime o smalldatetime de SQL Server, con la parte correspondiente a la fecha establecida en la fecha de inicio de Java de 1970/01/01 en la zona horaria del calendario suministrado.
