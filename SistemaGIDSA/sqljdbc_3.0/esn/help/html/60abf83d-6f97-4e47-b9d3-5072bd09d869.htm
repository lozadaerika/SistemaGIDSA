getDateTimeOffset(int) (SQLServerResultSet)

Este método se agregó en el controlador JDBC 3.0 de Microsoft SQL Server.

Recupera el valor de la columna designada como un objeto Clase DateTimeOffset en el lenguaje de programación Java según el índice del parámetro.

Sintaxis

public microsoft.sql.DateTimeOffset getDateTimeOffset(int columnIndex)

Parámetros

columnIndex

Ordinal de la columna.

Valor devuelto

Un objeto Clase DateTimeOffset.

Excepciones

SQLServerException

Notas

Puede actualizar un valor Clase DateTimeOffset con SQLServerResultSet.updateDateTimeOffset.
