Método position (java.sql.NClob, long)

Recupera la posición del carácter en la que el objeto NClob especificado searchstr aparece en este objeto NClob.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

long position(java.sql.NClob searchstr,
              long start)

Parámetros

searchstr

Un objeto NClob que se va a buscar.

start

La posición donde se empieza la búsqueda; la primera posición es 1.

Valor devuelto

La posición donde aparece la subcadena o -1 si no está presente. La primera posición es 1.

Excepciones

SQLServerException

Notas

El método position especifica este método position en la interfaz java.sql.NClob.
