Método getTime (int, java.util.Calendar)

Recupera el valor del parámetro designado como un objeto java.sql.Time en el lenguaje de programación Java según el índice de parámetro y mediante el uso del objeto Calendar determinado.

Sintaxis

public java.sql.Time getTime(int index,
                             java.util.Calendar cal)

Parámetros

index

Un valor int que indica el índice del parámetro.

cal

Un objeto Calendar.

Valor devuelto

Un objeto Time.

Excepciones

SQLServerException

Notas

El método getTime especifica este método getTime en la interfaz java.sql.CallableStatement.

Vea el gráfico titulado "Conversioens de métodos de captador" en Descripción de las conversiones de tipos de datos para consultar los tipos datos de SQL Server que se pueden recuperar con este método.
