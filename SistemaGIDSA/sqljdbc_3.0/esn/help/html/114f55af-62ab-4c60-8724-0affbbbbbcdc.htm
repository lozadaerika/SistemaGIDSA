Método valueOf (java.sql.Timestamp, int)

Crea un objeto DateTimeOffset que representa un punto cronológico dentro de un desplazamiento en concreto de GMT según un valor java.sql.Timestamp y un valor que indica el desplazamiento en minutos.

Sintaxis

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, int minutesOffset)

Parámetros

timestamp

Un valor java.sql.Timestamp.

minutesOffset

El desplazamiento en minutos.

Valor devuelto

Devuelve un objeto DateTimeOffset que representa un punto cronológico determinado por el objeto java.sql.Timestamp dentro de un desplazamiento especificado, en minutos, de GMT.
