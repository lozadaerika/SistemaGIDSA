Método setURL (SQLServerDataSource)

Establece la dirección URL que se utiliza para la conexión al origen de datos.

Sintaxis

public void setURL(java.lang.String url)

Parámetros

url

Un String que contiene la dirección URL.

Notas

Por razones de seguridad, no debería incluir la contraseña en la dirección URL que se proporciona al método setURL. Esto se debe a que los servidores de aplicación Java de otro fabricante muy a menudo muestran el conjunto de valores para la propiedad URL en la interfaz de usuario para la configuración del origen de datos. En su lugar, utilice el método setPassword para establecer el valor de la contraseña. Los servidores de aplicación Java no mostrarán las contraseñas que se establezcan en su origen de datos de la interfaz de usuario para la configuración.

Nota: Si no se llama al método setURL antes de llamar al método getURL, getURL devuelve el valor predeterminado de "jdbc:sqlserver://."
