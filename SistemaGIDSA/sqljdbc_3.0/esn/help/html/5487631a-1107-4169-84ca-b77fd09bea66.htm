Método setLastUpdateCount (SQLServerDataSource)

Establece un valor boolean que indica si la propiedad lastUpdateCount está habilitada.

Sintaxis

public void setLastUpdateCount(boolean lastUpdateCount)

Parámetros

lastUpdateCount

true si se habilita lastUpdateCount. En caso contrario, es false.

Notas

Si la propiedad lastUpdateCount está establecida en true, el controlador JDBC de Microsoft SQL Server devolverá solamente el último recuento de actualizaciones a partir de una instrucción SQL que se haya pasado al servidor. Si la propiedad lastUpdateCount está establecida en false, el controlador devolverá todos los recuentos de actualizaciones incluso los que se hayan devuelto por desencadenadores activados. Si no se establece la propiedad lastUpdateCount, el método getLastUpdateCount devuelve el valor predeterminado true.
