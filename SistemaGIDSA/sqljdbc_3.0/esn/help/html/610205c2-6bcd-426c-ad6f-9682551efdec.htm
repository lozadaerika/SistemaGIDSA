Método executeQuery (java.lang.String)

Ejecuta la instrucción SQL determinada y devuelve un objeto SQLServerResultSet único.

Sintaxis

public final java.sql.ResultSet executeQuery(java.lang.String sql)

Parámetros

sql

Un String que contiene una instrucción SQL.

Valor devuelto

Un objeto SQLServerResultSet.

Excepciones

SQLServerException

Notas

El método executeQuery especifica este método executeQuery en la interfaz java.sql.Statement.

Este método invalida el método executeQuery que se encuentra en la clase SQLServerStatement.

Si se llama a este método se producirá una excepción, ya que la instrucción SQL para el objeto SQLServerPreparedStatement se especificó cuando se creó el objeto.

Se produce la excepción SQLServerException si la instrucción SQL especificada genera algo que no sea un único objeto SQLServerResultSet.
