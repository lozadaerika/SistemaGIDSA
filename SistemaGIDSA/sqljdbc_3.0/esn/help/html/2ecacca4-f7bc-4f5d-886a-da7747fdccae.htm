Método cancelRowUpdates (ISQLServerResultSet)

Cancela las actualizaciones realizadas en la fila actual en este objeto SQLServerResultSet.

Sintaxis

public void cancelRowUpdates()

Excepciones

SQLServerException

Notas

El método cancelRowUpdates especifica este método cancelRowUpdates en la interfaz java.sql.ResultSet.

Se puede llamar a este método después de llamar a un método updater y antes de llamar al método updateRow para revertir las actualizaciones realizadas en una fila. Si no se han realizado actualizaciones o ya se ha llamado a updateRow, este método no tendrá efecto alguno.
