Método truncate (SQLServerBlob)

Trunca la longitud de un BLOB determinado.

Sintaxis

public void truncate(long len)

Parámetros

len

Nueva longitud del BLOB.

Excepciones

java.sql.SQLException

Notas

El método truncate especifica este método truncate en la interfaz java.sql.Blob.
