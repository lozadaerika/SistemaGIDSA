Trabajar con conjuntos de resultados

Si trabaja con los datos de una base de datos de SQL Server, uno de los métodos para manipular los datos consiste en usar un conjunto de resultados. El controlador JDBC de Microsoft SQL Server admite el uso de conjuntos de resultados mediante el objeto SQLServerResultSet. Si usa el objeto SQLServerResultSet, puede recuperar los datos devueltos por una instrucción SQL o un procedimiento almacenado, actualizar los datos según sea necesario y, a continuación, volver a almacenar los datos en la base de datos.

Además, el objeto SQLServerResultSet proporciona un método para desplazarse por las filas de datos, obtener o configurar los datos que contiene y establecer varios niveles de sensibilidad a los cambios en la base de datos subyacente.

Nota: Par obtener más información sobre la administración de los conjuntos de resultados, incluida la sensibilidad a los cambios, consulte Administrar conjuntos de resultados con el controlador JDBC.

En los temas de esta sección se describen las distintas formas en que puede usar un conjunto de resultados para manipular los datos de una base de datos de SQL Server.
