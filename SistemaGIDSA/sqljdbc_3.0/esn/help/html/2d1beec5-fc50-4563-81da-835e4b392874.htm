Método getLong (int) (ISQLServerResultSet)

Recupera el valor del índice de la columna designada en la fila actual de este objeto SQLServerResultSet como un valor long en el lenguaje de programación Java.

Sintaxis

public long getLong(int columnIndex)

Parámetros

columnIndex

Un valor int que indica el índice de la columna.

Valor devuelto

Un valor long .

Excepciones

SQLServerException

Notas

El método getLong especifica este método getLong en la interfaz java.sql.ResultSet.

Este método solamente se admite en los tipos de datos de SQL Server que pueden devolver de forma segura un valor entero como bigint, int, smallint, tinyint y bit. Si se utiliza este método en cualquier otro tipo de datos, provocará una excepción.
