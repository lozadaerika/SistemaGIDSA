Método getXAResource (SQLServerXAConnection)

Recupera un objeto SQLServerXAResource que utilizará el administrador de transacciones para administrar la participación de este objeto SQLServerXAConnection en una transacción distribuida.

Sintaxis

public javax.transaction.xa.XAResource getXAResource()

Valor devuelto

Un objeto XAResource.

Excepciones

java.sql.SQLException

Notas

El método getXAResource especifica este método getXAResource en la interfaz javax.sql.XAConnection.
