Método updateArray (int, java.sql.Array)

Actualiza la columna designada con un objeto Array según el índice de columna.

Sintaxis

public void updateArray(int columnIndex,
                        java.sql.Array x)

Parámetros

columnIndex

Un valor int que indica el índice de la columna.

x

Un objeto Array.

Excepciones

SQLServerException

Notas

El método updateArray especifica este método updateArray en la interfaz java.sql.ResultSet.
