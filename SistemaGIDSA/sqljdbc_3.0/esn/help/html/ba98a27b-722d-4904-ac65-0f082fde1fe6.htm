Método getBoolean (java.lang.String) (ISQLServerResultSet)

Recupera el valor del nombre de la columna designada en la fila actual de este objeto SQLServerResultSet como un valor boolean en el lenguaje de programación Java.

Sintaxis

public boolean getBoolean(java.lang.String columnName)

Parámetros

columnName

Un String que contiene el nombre de la columna.

Valor devuelto

Un valor boolean .

Excepciones

SQLServerException

Notas

El método getBoolean especifica este método getBoolean en la interfaz java.sql.ResultSet.

Este método se admite únicamente en los tipos de datos numéricos y de caracteres. Convierte valores "1", 1 y "true" en true y valores "0", 0 y "false" en false. Para el resto de los valores no se ha definido el comportamiento.
