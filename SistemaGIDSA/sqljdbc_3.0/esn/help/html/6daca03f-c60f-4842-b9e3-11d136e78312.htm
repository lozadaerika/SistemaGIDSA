Método updateNString (java.lang.String, java.lang.String)

Actualiza la columna designada con un valor String mediante el uso de la etiqueta de columna especificada.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public void updateNString(java.lang.String columnLabel,
                          java.lang.String nString)

Parámetros

columnLabel

Un String que contiene la etiqueta de columna.

nString

Un objeto String .

Excepciones

SQLServerException

Notas

El método updateNString especifica este método updateNString en la interfaz java.sql.ResultSet.

Este método pasa el String de Java a las columnas nchar, nvarchar(max), ntexty xml seleccionadas. Si se utiliza este método en otras columnas de tipo de datos, se producirá una excepción.
