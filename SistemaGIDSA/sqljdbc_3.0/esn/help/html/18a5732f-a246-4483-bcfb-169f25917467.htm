Método free (SQLServerBlob)

Este método libera al objeto BLOB y a los recursos que contiene.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public void free()

Excepciones

SQLServerException

Notas

El método free especifica este método free en la interfaz java.sql.Blob.
