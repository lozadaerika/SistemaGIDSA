Clase SQLServerXAResource

Representa un XAResource para la administración de transacciones distribuidas XA.

Paquete: com.microsoft.sqlserver.jdbc

Extiende: java.lang.Object

Implementa: javax.transaction.xa.XAResource

Sintaxis

public class SQLServerXAResource

Notas

Las transacciones XA se implementan en SQL Server con Microsoft DTC (Coordinador de transacciones distribuidas). La clase SQLServerXAResource efectúa llamadas a un archivo extendido dll de SQL Server denominado sqljdbc_xa.dll, que se interrelaciona con DTC. Las llamadas XA que recibe SQLServerXAResource (XA_START, XA_END, XA_PREPARE, etc.) están asignadas a las llamadas correspondientes a las funciones DTC.
