Método setTrustServerCertificate (SQLServerDataSource)

Establece un valor boolean que indica si la propiedad trustServerCertificate está habilitada.

Sintaxis

public void setTrustServerCertificate(boolean trustServerCertificate)

Parámetros

trustServerCertificate

true si se debe confiar de forma automática en el certificado de Capa de sockets seguros (SSL) del servidor cuando el nivel de comunicación se cifre con SSL. En caso contrario, es false.

Notas

Si la propiedad trustServerCertificate se establece en true, se confía automáticamente en el certificado SSL de SQL Server, cuando el nivel de comunicación se cifre con SSL. Es decir, el controlador JDBC de Microsoft SQL Server no validará el certificado SSL de SQL Server. El valor predeterminado es false.

Si la propiedad trustServerCertificate se establece en false, el controlador JDBC de Microsoft SQL Server validará el certificado SSL del servidor.
