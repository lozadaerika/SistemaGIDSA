Método setQueryTimeout (ISQLServerStatement)

Establece el número de segundos que esperará el controlador antes de que se ejecute un objeto SQLServerStatement según el número de segundos determinado.

Sintaxis

public final void setQueryTimeout(int seconds)

Parámetros

seconds

Un valor int que indica el número de segundos que transcurrirán; se establece en 0 si no hay límite alguno.

Excepciones

SQLServerException

Notas

El método setQueryTimeout especifica este método setQueryTimeout en la interfaz java.sql.Statement.
