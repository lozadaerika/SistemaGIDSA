Método setTimestamp (int, java.sql.Timestamp, java.util.Calendar)

Establece el parámetro designado para los valores timestamp y calendar especificados.

Sintaxis

public final void setTimestamp(int n,
                               java.sql.Timestamp x,
                               java.util.Calendar cal)

Parámetros

n

Un valor int que indica el número del parámetro.

x

Un objeto Timestamp.

cal

Un objeto Calendar.

Excepciones

SQLServerException

Notas

El método setTimestamp especifica este método setTimestamp en la interfaz java.sql.PreparedStatement.
