Método setClientInfo (java.lang.String, java.lang.String)

Establece el valor de la propiedad de información de cliente especificada.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public void setClientInfo (java.lang.String name,
                           java.lang.String value)

Parámetros

name

Un String que contiene el nombre de la propiedad de información de cliente que se va a establecer.

value

Un String que contiene el valor según el que se va a establecer la propiedad de información de cliente.

Excepciones

SQLServerException

Notas

El método setClientInfo especifica este método setClientInfo en la interfaz java.sql.Connection.

El controlador JDBC de Microsoft SQL Server no admite propiedades de información de cliente. En la versión 2.0 del controlador JDBC, este método genera una advertencia para una propiedad. Las aplicaciones deberían utilizar el método getWarnings de la clase SQLServerConnection para recuperar una advertencia.
