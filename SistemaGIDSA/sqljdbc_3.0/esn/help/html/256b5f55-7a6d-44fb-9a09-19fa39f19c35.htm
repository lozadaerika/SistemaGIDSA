Método setClob (java.lang.String, java.sql.Clob)

Establece el parámetro designado para el objeto Clob especificado.

Nota: Esta característica se incorpora a partir de la versión 2.0 del controlador JDBC de Microsoft SQL Server.

Sintaxis

public final void setClob(java.lang.String parameterName,
            java.sql.Clob x)

Parámetros

parameterName

Un String que contiene el nombre del parámetro.

x

Un objeto Clob.

Excepciones

SQLServerException

Notas

El método setClob especifica este método setClob en la interfaz java.sql.CallableStatement.
