Método setUser (SQLServerDataSource)

Establece el nombre de usuario que se utiliza para la conexión al origen de datos.

Sintaxis

public void setUser(java.lang.String user)

Parámetros

user

Un String que contiene el nombre de usuario.

Notas

El método setUser establece el nombre de usuario que se utilizará para conectar a SQL Server. Si no se establece un valor de nombre de usuario, el método getUser devuelve el valor predeterminado o NULL.
