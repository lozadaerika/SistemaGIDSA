Método getDouble (java.lang.String) (ISQLServerResultSet)

Recupera el valor del nombre de la columna designada en la fila actual de este objeto SQLServerResultSet como un valor double en el lenguaje de programación Java.

Sintaxis

public double getDouble(java.lang.String columnName)

Parámetros

columnName

Un String que contiene el nombre de la columna.

Valor devuelto

Un valor double .

Excepciones

SQLServerException

Notas

El método getDouble especifica este método getDouble en la interfaz java.sql.ResultSet.

Este método devuelve todos los tipos de datos basados en número con fidelidad double de Java.
